Control de Lotes

Interfaz sencilla para gestionar lotes por sucursal

Control de inventario por lote

Aplicación multiplataforma con sincronización a GitHub, exportación a CSV y gráficas integradas.

Windows Android v1.0.5

Características

  • Registro rápido de lotes con fecha, variedad y cantidades
  • Sincronización con GitHub para backups en la nube
  • Gráficos de progreso por variedad
  • Exportación a CSV
