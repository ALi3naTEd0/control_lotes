Control de Lotes

Gestión sencilla de lotes por sucursal — crear, editar, auditar y sincronizar con GitHub.

Ver README Contacto
Interfaz GUI

Pestañas para crear lotes, agregar variedades, editar y listar con exportación a PDF/XLSX.

Sincronización GitHub

Descarga y subida del CSV principal (`lotes_template.csv`) vía GitHub API y gestión de `sha`.

Backups automáticos

Backups timestamped en `registros/` y opción de restaurar el último backup disponible.

Gráficos y resúmenes

Radar, pastel y barras para visualizar distribución por sucursal, etapa y ubicación.

Nota: Si el logo no se muestra correctamente en GitHub Pages, copia `assets/icon.png` a `docs/assets/icon.png` (o actualiza la ruta de la imagen en este archivo). Para ayuda con despliegue en GitHub Pages, puedo agregar automáticamente los pasos o un pequeño script de build.
